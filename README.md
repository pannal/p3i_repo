# p3i Kodi repo

Kodi addon repository for the [p3i CoreELEC build](https://github.com/pannal/CoreELEC) (aml-4.9-21.3 lineage).

Hosts binary and Python addons built specifically against p3i's Kodi + FFmpeg sysroot. Not intended for use on stock CoreELEC builds — binaries here are ABI-matched to p3i firmware and may fail to load elsewhere.

## Install

Download `repository.p3i-*.zip` from [p3irepo.pm4k.eu](https://p3irepo.pm4k.eu/) and install via Kodi's *Add-ons → Install from zip file*.

After that, p3i appears under *Install from repository*.
